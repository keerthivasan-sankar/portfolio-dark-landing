# Keerthivasan S — Portfolio

A dark-themed, single-page portfolio landing page built with **React + Vite + Tailwind CSS + TypeScript + GSAP + Framer Motion + hls.js**.

## Live Preview

Open [`portfolio-preview.html`](./portfolio-preview.html) in any browser to see a static preview, or run the React app locally.

## Tech Stack

| Technology | Purpose |
|---|---|
| React 18 | UI framework |
| Vite | Build tool & dev server |
| TypeScript | Type safety |
| Tailwind CSS | Utility-first styling |
| Framer Motion | React animations |
| GSAP + ScrollTrigger | Scroll-driven animations |
| hls.js | HLS video streaming |

## Sections

1. **Loading Screen** — Animated counter with rotating words
2. **Hero** — HLS background video, GSAP entrance, cycling roles
3. **Selected Works** — Real projects from resume with descriptions & tech stacks
4. **Skills** — Languages, Web & Data, Tools, Soft Skills
5. **Experience** — UI/UX Designer, B.Tech CSE, Higher Secondary
6. **Journal** — Blog-style entries
7. **Stats** — Real metrics (4+ Projects, 95% ML Accuracy, 7.0 CGPA, 8 Days Fastest Build)
8. **Contact Footer** — HLS video, GSAP marquee, email CTA, social links

## Getting Started

```bash
# Clone the repo
git clone https://github.com/keerthivasan-sankar/portfolio-dark-landing.git
cd portfolio-dark-landing

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Design System

- **Fonts:** Inter (body), Instrument Serif (display)
- **Colors:** Near-black `#0A0A0A`, warm white `#F5F5F5`, accent `#89AACC` → `#4E85BF`
- **Animations:** 60fps, `prefers-reduced-motion` friendly
